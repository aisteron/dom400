стеновой комплект

  1. Сборка стенового комплекта осуществляется на сухие “нагели” с шагом от 0.5м до 1.3м
  2. Равномерность сборки контролируется на каждом ряду
  3. Сборка ведется на межвенцовый утеплитель политерм, он разжимается при усадке (конопатка не потребуется)
  4. В местах стыковки бревен они соединяются “скобами”и накрываются “перерубом” для исключения мостиков холода.
  5. Сборка начинается с гидроизоляции фундамента, далее закладная доска 50 мм*200 мм (лиственница, сосна), затем полубревно, и начинается сборка всей стопы сруба

фронтоны

1. Бревенчатые фронтоны. Скрепляются “нагелями” и пружинными узлами “сила” с целью предотвращения зависания бревен

2. Каркасные фронтоны. Вместо бревенчатых фронтонов можно использовать “каркасные” с целью экономии денежных средств

виды крышы

Проектируем и устанвливаем любые крыши с расширенной гарантией!

Двухскатная крыша -собирается простой конструктив -закрепляется на скользящие элементы (для усадки) -создается дополнительное пространство

Крыша с ендовами -сложный конструктив крыши -закрепляется на скользящие элементы (для усадки) -создается дополнительное пространство -возможно только временное финишное покрытие сразу

Вольмовая крыша -устанавливается без фронтонов -закрепляется без скользящих элементов (не требует усадки)

Двухскатная ломанная крыша -устанавливается с каркасными фронтонами -закрепляется без скользящих элементов (не требует усадки) -создает дополнительное пространство усадки для проживания
